# 禁ギャンブル記録アプリ（Gamble-Free Tracker）

「今日も我慢できた！」を記録する、React製の禁ギャンブル支援アプリです。

## 🛠️ 主な機能
- 毎日ボタンを押して継続日数をカウント
- 継続日数に応じて「見習い」〜「レジェンド」までランク表示
- 記録はグラフで可視化（Chart.js）
- LocalStorageによるデータ保持
- スマホ対応／ダークモード対応
